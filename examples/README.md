# Examples

This directory contains small public fixtures and generated sample outputs.

Do not add private project data, credentials, raw reports, or local memory files.

- `fixtures/codex-basic`: synthetic Codex-style workspace for scanner testing.
- `fixtures/claude-code-basic`: synthetic Claude Code workspace with `CLAUDE.md`, `.claude/rules`, `.claude/skills`, and `.claude/commands`.
- `fixtures/cursor-basic`: synthetic Cursor workspace with `AGENTS.md`, `.cursor/rules`, `.cursor/skills`, `.cursor/commands`, and MCP config.
